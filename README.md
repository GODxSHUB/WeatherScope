# 🌤️ WeatherScope

WeatherScope is a sleek, modern, and responsive weather dashboard that provides real-time atmospheric data. Built with a focus on high-end UI/UX, it utilizes the OpenWeatherMap API to deliver precise weather updates with a "Glassmorphism" aesthetic.

## ✨ Features
* **Real-time Weather:** Accurate temperature, humidity, wind speed, and pressure.
* **Dynamic Visuals:** Weather-specific emojis and descriptive labels (e.g., "Mostly Cloudy," "Very Humid").
* **Quick Search:** Pre-defined chips for major cities for instant access.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Modern Stack:** Pure HTML5, CSS3 (with Syne & DM Sans typography), and Vanilla JavaScript.
